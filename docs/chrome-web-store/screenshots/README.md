**Upload:** Chrome Web Store Developer Dashboard → your extension → **Store listing** → **Screenshots** (not inside the `.zip` package).

**Current file sizes (pixels):** popup images ~446–451×520–527; dashboard images ~1024×485–487. Chrome’s docs often cite **1280×800** or **640×400** as ideal. If the upload form rejects a file, recapture the **dashboard** full browser window at that size, or place the popup on a neutral background in an editor and export to **1280×800**.
